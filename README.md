# LaTeX Starter Template

A simple starter template for $\LaTeX$. Create document section in `.tex/` folder.

## Setup

Install [MacTex](https://www.tug.org/mactex/) for OSX (the package size is about 6 GB):

```
brew install --cask mactex
```

## Usage

```
make prettier
make
make clean
open main.pdf
```
